# CLM Tool

Campaign data visualizer

## Installation


```
Clone repo: git clone https://github.com/ahetawal/clmtool.git;
Cd: cd clmttool
Install dependencies: npm install
Start server: DEBUG=clmtool:* npm start
```

## Usage

1. Signin
2. View your data
